## ⚙️ Nasıl Derlenir ve Çalıştırılır?

```bash
# Derleme
$ g++ dosya_adi.cpp -o program_adi
# Çalıştırma
$ ./program_adi
```

Örnek:
```bash
g++ 1_helloWorld.cpp -o hello
./hello
```
